CSS

Some of the CSS used to style this site!

CSS stands for cascading style sheets. It's used to style HTML in a robust way. Styles made in a CSS file can be applied to many different web pages. CSS is what makes the internet look the way it does.

Good things to remember

  • Classes can be reused on a page, but ids cannot
  • You can create styles for divs
