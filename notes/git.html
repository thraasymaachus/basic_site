GIT


Jason Long, CC BY 3.0, via Wikimedia Commons

Git is a version controll system, commonly used in development. A git repository (repo) is made up of snapshots that a developer has commited to the repo. Git is a very useful tool for looking back over code, and keeping a backup.

Good things to remember

  • git can be used via the command line, or a GUI
    • git on the cmd gives potential to much more powerful operations
  • There is a
    • Working directory, where you edit code directly
    • Staging area, where files sit before you commit them
    • Repository, which holds the project's snapshots, and tree hashes
