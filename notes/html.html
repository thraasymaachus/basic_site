HTML

Some of the HTML used to build this site!

HTML is the building material of all sites on the world wide web. It tells your browser what information to display, and where. HTML is written like code, but is not in fact a programming language. It stands for HyperText Markup Language. HTML never involves executing commands.

Good things to remember

  • w3schools.com is a great reference for learning HTML
  • The creation of HTML can be automated using python, or website builders
  • HTML doesn't look very good on it's own. It often works alongside
    • Javascript, to add interactivity and processing
    • CSS, in order to style it
  • escape codes can be used to write special characters
    • e.g. < and > are written &lt and &gt respectively
